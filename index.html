📊 Cara Integrasi Google Sheets

Ikuti langkah berikut untuk menghubungkan WalletUS ke Google Sheets kamu. Data akan tersinkron secara otomatis via Google Apps Script.

1 Buat Google Sheet Baru

Buka sheets.google.com → buat spreadsheet baru → beri nama "WalletUS Data". Buat 3 sheet tab: Transactions, Accounts, Summary

2 Pasang Apps Script

Di Google Sheet → Extensions → Apps Script → paste kode berikut:

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet();
  var data = JSON.parse(e.postData.contents);
  
  if (data.type === 'transaction') {
    var txSheet = sheet.getSheetByName('Transactions');
    txSheet.appendRow([
      data.date, data.desc, data.category,
      data.account, data.txType, data.amount, data.note
    ]);
  } else if (data.type === 'account') {
    var accSheet = sheet.getSheetByName('Accounts');
    // Update or insert account balance
    var rows = accSheet.getDataRange().getValues();
    var found = false;
    for (var i = 0; i < rows.length; i++) {
      if (rows[i][0] === data.name) {
        accSheet.getRange(i+1, 2).setValue(data.balance);
        found = true; break;
      }
    }
    if (!found) accSheet.appendRow([data.name, data.balance, data.color]);
  }
  
  return ContentService
    .createTextOutput(JSON.stringify({status:'ok'}))
    .setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  return ContentService
    .createTextOutput(JSON.stringify({status:'healthy'}))
    .setMimeType(ContentService.MimeType.JSON);
}
3 Deploy sebagai Web App

Klik Deploy → New Deployment → Web App. Set "Execute as: Me" dan "Who has access: Anyone". Salin URL deployment-nya.
