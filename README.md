# FinanceOS 💰
> Personal Money Command Center — catat pengeluaran, pemasukan & transfer antar akun

## 🚀 Deploy ke GitHub Pages (5 menit)

### Langkah 1: Buat Repository
1. Buka **github.com** → klik **New repository**
2. Nama repo: `financeos` (atau nama lain)
3. Set ke **Public**
4. Klik **Create repository**

### Langkah 2: Upload File
Upload semua file berikut ke repo:
```
index.html
manifest.json
sw.js
icon-192.png
icon-512.png
README.md
```

**Cara upload:**
- Di halaman repo → klik **Add file** → **Upload files**
- Drag & drop semua file → klik **Commit changes**

### Langkah 3: Aktifkan GitHub Pages
1. Di repo → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: `main` / `(root)`
4. Klik **Save**
5. Tunggu ~1 menit → URL muncul: `https://USERNAME.github.io/financeos`

## 📱 Install sebagai Shortcut HP

### Android (Chrome)
1. Buka URL di Chrome
2. Tap menu **⋮** (tiga titik) → **Add to Home screen**
3. Beri nama → **Add**
4. Ikon FinanceOS muncul di home screen!

### iPhone (Safari)
1. Buka URL di **Safari** (wajib Safari, bukan Chrome)
2. Tap ikon **Share** (kotak dengan panah atas)
3. Scroll ke bawah → **Add to Home Screen**
4. Beri nama → **Add**
5. Ikon FinanceOS muncul di home screen!
